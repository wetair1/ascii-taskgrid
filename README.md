# ascii-taskgrid

Small curses task board for the terminal.

## Features

- Add tasks from inside the TUI
- Toggle tasks as done
- Delete tasks
- Saves data to your home directory
- Pure Python stdlib

## Usage

```bash
python3 main.py
```

## Controls

- `a`: add task
- `space`: toggle done
- `d`: delete task
- `j` / Down: move down
- `k` / Up: move up
- `q`: save and quit
